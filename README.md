## Estructura

- `main` → Rama base sin ningún proyecto específico.
- `Guia01` → Proyecto correspondiente a la Guía 1.
- `Guia02` → Proyecto correspondiente a la Guía 2.
- `Guia03` → Proyecto correspondiente a la Guía 3.
- etc...
